# Roadmap del onboarding-agent

## Fase 1

- Crear expediente desde plantilla
- Registrar datos clave manualmente
- Revisar faltantes críticos
- Generar resumen operativo

## Fase 2

- Actualizaciones por sección (contactos, dominios, plataformas, seguridad)
- Recalcular completitud
- Proponer tareas para Paperclip

## Fase 3

- Importar inventario de equipos desde JSON
- Normalizar hardware, software y seguridad
- Vincular equipos a clientes y usuarios

## Fase 4

- Rutinas Paperclip para seguimiento automático
- Alertas por vencimiento y faltantes críticos
- Dashboard resumido por cliente
